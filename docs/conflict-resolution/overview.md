# Fork 增强功能概览

## 新增功能

### Kiro (AWS CodeWhisperer) 配额显示
- 配额管理页面显示 Kiro 配额
- 认证文件页面支持配额刷新
- 显示基础配额和免费试用配额

### GitHub Copilot 配额显示
- 配额管理页面显示 Copilot 配额
- 认证文件页面支持配额刷新
- 支持 Free/Pro 和 Business/Enterprise 格式

## 代码标记

所有 Fork 增强代码都有注释标记：

```typescript
// Fork 增强: Kiro 和 Copilot 配额支持
```

或分隔线：

```typescript
// ============================================================================
// Fork 增强: Kiro 和 Copilot 配额支持
// ============================================================================
```

## 合并策略

1. **上游代码优先** - 接受上游的结构变化
2. **追加我们的代码** - 在上游代码后面添加 Kiro/Copilot 支持
3. **保持注释标记** - 便于下次识别
